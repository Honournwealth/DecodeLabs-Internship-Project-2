# DecodeLabs-Internship-Project-2
Project 2: Exploratory Data Analysis (EDA)
📊 Overview

This project focuses on performing Exploratory Data Analysis (EDA) on a cleaned transactional dataset using Python.

The objective is to identify patterns, trends, correlations, distributions, and outliers through statistical analysis and data visualization.

🎯 Objectives
Calculate descriptive statistics
Analyze trends and distributions
Detect outliers
Generate visual insights
Understand customer purchasing behavior
🛠 Tools Used
Python
Pandas
NumPy
Matplotlib
Seaborn
📂 Dataset Summary
Rows: 891
Columns: 14
Missing Values: 0
Date Range: 2023 – 2025
🔍 Key Analysis Steps
1. Data Inspection
Used df.head(), df.info(), and df.describe()
2. Descriptive Statistics
Calculated:
Mean
Median
Count
Standard Deviation
Minimum & Maximum values
3. Distribution Analysis
Created histograms for numerical variables
4. Outlier Detection
Used boxplots to identify extreme values
5. Correlation Analysis
Generated correlation heatmaps for numerical relationships
6. Trend Analysis
Visualized sales trends over time using line charts
7. Categorical Analysis
Created countplots for:
Payment Methods
Order Status
📊 Key Insights
Average customer purchase quantity was approximately 3 items
Average transaction value was about 1057.64
Most carts contained between 4–7 items
Some high-value transactions were detected as outliers
Dataset showed trends across multiple years
📈 Visualizations Generated
Histograms
Boxplots
Correlation Heatmap
Countplots
Sales Trend Line Chart
Top Products Bar Chart
